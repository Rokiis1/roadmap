# Summary

In this level, we learned the basics of **control flow** in Python. Control flow allows a program to make decisions and repeat actions instead of always running line by line.

We started with **conditional statements** using `if`, `else`, and `elif`. These statements allow a program to run different blocks of code depending on whether a condition is **True** or **False**.

Next, we explored **loops**, which allow code to run multiple times. We learned how the `for` loop works with **iterable data types**, such as strings, and how the `range()` function creates a sequence of numbers that can be used in loops.

We also learned how the `while` loop works and how it repeats code as long as a condition remains **True**.
