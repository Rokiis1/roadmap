# Content of Python Environment 1 level

When working in your **local environment**, you have Python installed on your machine. There are two main ways to work locally:

- **Interactive Mode Programming:** You can launch Python **interactively** to test snippets of code. To get started, open your **terminal** or **command prompt**, or use **IDLE**, type `python` or `py`, and press `Enter`. Once the interpreter starts, you'll see the `>>>` prompt where you can type Python commands and view their output immediately.

    This mode, known as the **REPL (Read-Eval-Print-Loop)**

    1. **Reading** your input.
    2. **Evaluating** the command.
    3. **Printing** the result.
    4. **Looping** back to wait for the next command.

    To exit the interactive mode, type `quit()` or `exit()`.

- **Script Mode Programming:** Another way to work locally is by creating Python scripts. This involves writing your code in a text editor, saving the file with a `.py` extension and then executing it from your terminal. By convention, `main.py` is the file where you start create applciation. Navigate to your script's directory and run it by typing `python filename.py`, `python3 filename.py` on some systems, or `py filename.py`.
