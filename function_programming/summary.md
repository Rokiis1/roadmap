## Function Programming Level 1

### Summary Python Function Programming Level 1

In this level, we learned the fundamentals of **Python function programming**, focusing on how functions help organize code, reuse logic, and structure programs more effectively.

We began by learning the difference between **function definition and function call**, understanding that defining a function describes what it does, while calling a function actually executes its code. We also explored **parameters and arguments**, learning how functions receive input values when they are called.

Next, we focused on the important distinction between **`return` and `print`**. We learned that `print()` is mainly used for displaying output, while `return` sends a value back from a function so it can be reused elsewhere in the program. This concept is essential for writing reusable and composable functions.

Building on this idea, we introduced **function composition**, where the return value of one function is passed directly into another function call. We saw that function composition only works correctly when functions return values, and we examined common errors that occur when `print()` is used instead of `return`, including Python’s automatic return of `None`.

We then explored **control flow inside functions**, learning how conditional statements and loops allow functions to make decisions and handle more complex logic. This made it possible to write functions that behave differently depending on their input.

After that, we worked with **functions and collections**, using membership and identity operators inside functions to check whether values exist in a collection or whether two variables refer to the same object in memory.

Finally, we introduced several **built-in numeric utility functions**, such as `abs()`, `round()`, `min()`, `max()`, `sum()`, and `pow()`. These functions demonstrated how Python provides ready-made tools that can be combined with user-defined functions to write clearer and more efficient code.

By the end of this level, you should be comfortable **defining functions**, **passing data into them**, **returning values**, and **combining functions** to solve practical programming problems.
